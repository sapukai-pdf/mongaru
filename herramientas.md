Simplemente empieza a escribir.

Manuskript es una herramienta de código abierto para escritores. El programa corre en GNU/Linux, Mac OS X, y Windows.

#### Características

Manuskript provee un entorno rico para ayudar a los escritores en su primer borrador, para luego refinar y editar su obra final.

Con Manuskript puedes:

* Hacer crecer a tu premisa a partir de una sola oración, un párrafo, a un resumen entero.
* Crear personajes.
* Concebir tramas.
* Construir esbozos.
* Escribir sin distracciones.
* Construir mundos.
* Hacer seguimiento de elementos.
* Editar y re-organizar capítulos y escenas.
* Ver una linea argumental.
* Escribir con plantillas de ficción o no-ficción y modos de escritura.
* Importar y exportar documentos con formatos como HTML, ePub, OpenDocument, DocX, y más.
  
Adicionalmente, Manuskript te puede ayudar de otras maneras con un corrector de ortografía, resaltador de markdown, analizador de frecuencias, y guardado automático en formato de texto plano.

## Aplicaciones para tomar notas

<details>
<summary>

### [Joplin](https://joplinapp.org/help/#installation)

</summary>


Joplin es una aplicación de código abierto gratuita para hacer notas y listas de tareas, la cual puede manejar una gran cantidad de notas organizadas en cuadernos. Se puede hacer búsqueda de las notas, copiarlas, etiquetarlas y modificarlas, ya sea desde la aplicación o desde tu propio editor de texto. Las notas están en el formato Markdown.

Las notas exportadas de Evernote se pueden importar a Joplin, incluido el contenido formateado (que se convierte a Markdown), recursos (imágenes, archivos adjuntos, etc.) y metadatos completos (geolocalización, hora actualizada, hora creada, etc.). También se pueden importar archivos Markdown simples. 

Las notas se pueden sincronizar mediante el cifrado de extremo a extremo con varios servicios en la nube, incluidos Nextcloud, Dropbox, OneDrive y Joplin Cloud .

La búsqueda de texto completo está disponible en todas las plataformas para encontrar rápidamente la información que necesita. La aplicación se puede personalizar usando complementos y temas, y también puede crear fácilmente los suyos propios.

La aplicación está disponible para Windows, Linux, macOS, Android e iOS. Un Web Clipper , para guardar páginas web y capturas de pantalla desde su navegador, también está disponible para Firefox y Chrome .

### Gestores de tareas


<details>
<summary>

### [Logseq](https://logseq.com)

</summary>


Logseq es una plataforma para compartir y gestionar conocimientos.  Se centra en la privacidad, la longevidad y el control del usuario.  Aviso: el código de backend será de código abierto tan pronto como estemos seguros de que el servicio de backend cumple con los estándares de seguridad. 

El servidor nunca almacenará ni analizará sus notas privadas.  Sus datos son archivos de texto sin formato y actualmente admitimos los modos Markdown y Emacs Org (pronto se agregarán más). 

En el improbable caso de que el sitio web esté caído o no se pueda mantener, sus datos son y siempre serán suyos. 

</details>
